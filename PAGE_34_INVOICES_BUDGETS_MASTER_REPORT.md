# تقرير الصفحة 34 - الفواتير والتحصيل والميزانيات التقديرية

## الملخص
تم تنفيذ صفحة 34 لواجهة الفواتير والتحصيل مع الميزانيات التقديرية ونموذج الميزانية التفصيلي وتحليل الانحراف، وربط البيانات بقاعدة البيانات وعرضها في الجداول والبطاقات.

## المكونات الرئيسية
- مؤشرات الفواتير والتحصيل: الإجمالي، المدفوع، المتبقي، نسبة التحصيل، المتأخر، المسدد، خطط السداد، متوسط قيمة الفاتورة.
- جدول الفواتير مع أزرار السداد وخطة الدفع.
- واجهة الميزانيات التقديرية: إجمالي الميزانية، الفعلي، الانحراف، نسبة الالتزام.
- نموذج الميزانية التقديرية التفصيلي (إيرادات/مصروفات/تدفقات).
- ميزانية رأس المال (افتتاح فروع، تطوير منصات، شراء أصول).
- تحليل الانحراف وجدوله.
- جداول الميزانيات وسطـور الميزانية.

## الأزرار
