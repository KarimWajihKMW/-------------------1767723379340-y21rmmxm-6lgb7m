# تقرير الصفحة 35 - المخطط التنفيذي للنظام

## الملخص
تم تنفيذ صفحة 35 لعرض المخطط التنفيذي للنظام مع تعريف الوحدات والجداول والتدفقات الأساسية وربطها بالبيانات الحية من قاعدة البيانات.

## المكونات الرئيسية
- مؤشرات: عدد الوحدات، عدد الجداول الأساسية، عدد التدفقات، عدد السجلات الحية.
- عرض الوحدات الأساسية داخل النظام.
- جدول الجداول الجوهرية مع عدد السجلات.
- عرض التدفقات الأساسية للنظام.
- جداول البيانات الحية: العملاء، الفواتير، المدفوعات، خطط السداد، شجرة الحسابات، القيود وسطور القيود، الميزانيات والانحرافات، تقييمات المخاطر، التوقعات الذكية.

## الأزرار
- تحديث البيانات
- تنزيل ملف بيانات
- تنزيل ملف جدولي
- طباعة
- نسخ المخطط (محاكاة)
